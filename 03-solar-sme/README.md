# Solar SME

## Purpose

This folder is for CESES solar opportunities with small and medium businesses, commercial buildings, factories, farms, schools, and other organizations.

## Key Questions

- Which SME customers have high electricity costs?
- What solar package is easiest to sell and deliver?
- What savings can CESES clearly show to customers?
- Which partners are needed for engineering, equipment, installation, and maintenance?

## Required Documents

- Target customer list
- Solar offer description
- Site survey checklist
- Basic savings calculator
- Equipment and installer partner list
- Pilot project notes

## Next Actions

- Choose 2 or 3 target customer types.
- Draft a simple solar offer for SMEs.
- Build a basic site survey checklist.
- Identify possible pilot customers.
